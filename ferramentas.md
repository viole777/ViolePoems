# HTML: quais elementos estão sendo usados
<nav> para fazer o menu
<a href=""> para colocar os links
<h3> para colocar o titulo da página index
<div> para fazer o container
<p> para fazer o parágrafo do container e deixar organizado
<strong> para deixar a parte importante do container em negrito
<header> é para fazer o o cabeçalho/menu da pagina
<br> quebrar a linha
<img> adicionar a imagem
<footer> para fazer o rodapé da página
<form> para criar um formulário web

# JS: há algum recurso javascript?

O site usa JavaScript para:

Capturar os valores inseridos nos campos do formulário (título e poema).

Validar o formulário:
✔ impede envio vazio
✔ impede poema grande demais

Manipular o DOM criando elementos dinamicamente (divs, títulos, parágrafos).

Exibir o poema publicado na página sem recarregar.

Usar eventos de clique com addEventListener().

Usar preventDefault() para impedir o envio automático do formulário.
